# Agent Decision Log

This directory captures agent-generated decisions that affect implementation scope, architecture, or release readiness.

## Naming

Use `AGDEC-SPR<nn>-YYYY-MM-DD-<agent>.md`.

## Required Fields

1. Decision ID
2. Agent
3. Sprint ID
4. Related Story/Task IDs
5. Decision summary
6. Rationale
7. Risk assessment
8. Approval status
