# Release Checklist

## Pre-Release

1. All committed stories meet DoD.
2. Test suites pass.
3. Security scan complete.
4. Drift checks pass.
5. MGT version linkage complete.
6. Blocking CI gate status is green.

## Release Bundle

1. Code delta reference.
2. MGT version.
3. Session trace references.
4. Audit report.
5. Story and task linkage references.

## Post-Release

1. Release notes published.
2. Metrics snapshot captured.
3. Retrospective action items logged.
4. Advisory gate exceptions and expirations reviewed.
