# PR Checklist Template

## Required Linkage

1. Story ID(s) linked
2. Task ID(s) linked
3. Session log reference linked
4. ADR reference linked (if architecture changed)

## Blocking Checks

1. Tests pass
2. Security scan has no critical/high findings
3. Traceability fields complete

## Advisory Checks

1. Backlog refinement notes updated if scope changed
2. Sprint log updated
3. Metrics snapshot impacted fields updated
