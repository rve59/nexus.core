# Sprint Execution Workspace

This directory stores sprint records in sprint-scoped folders.

## Structure

- `SPR-<id>-<name>/plan.md`: Sprint planning baseline and commitments.
- `SPR-<id>-<name>/log.md`: Daily execution notes and scope decisions.
- `SPR-<id>-<name>/retrospective.md`: Sprint retrospective with actions.
- `SPR-<id>-<name>/velocity_snapshot.json`: Planned vs completed metrics.
- `templates/`: Reusable sprint templates.

## Naming

Use `SPR-00-bootstrap`, `SPR-01-intent-core`, and increment sequentially.
