# Session Logs

Operational session logs live here.

## Naming Convention

Use `SL-SPR<nn>-YYYY-MM-DD-HHmm-topic.md`.

Example: `SL-SPR01-2026-03-25-1400-orchestrator-review.md`

## Required Metadata

1. Session ID
2. Date and time
3. Sprint ID
4. Story IDs
5. Decisions made
6. Risks and blockers
7. Next actions
