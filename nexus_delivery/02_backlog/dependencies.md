# Backlog Dependencies

Track cross-feature and cross-sprint dependencies.

## Dependency Record

1. Dependency ID: DEP-<seq>
2. Source: <Epic/Feature/Story/Task>
3. Depends on: <Epic/Feature/Story/Task>
4. Type: technical|process|external
5. Owner
6. Due sprint
7. Status: open|in_progress|resolved|blocked
8. Risk if unresolved
