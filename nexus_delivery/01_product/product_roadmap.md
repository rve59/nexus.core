# Product Roadmap (Initial)

## Phase R0 - Foundation (Sprint 0)

- Delivery operating system and governance artifacts.
- FastAPI service boundary and architecture baseline.
- Initial backlog and sprint framework.

## Phase R1 - Intent Core (Sprints 1-2)

- MGT schema, validation, and versioning.
- Project lifecycle APIs and session trace persistence.
- Initial Workbench integration for intent operations.

## Phase R2 - Orchestration Core (Sprints 3-4)

- Consultant loop and dependency-based synthesis orchestration.
- Auditor rules and drift detection service.
- Retry and HITL escalation mechanics.

## Phase R3 - Integrations (Sprints 5-6)

- GAS connector and ingestion flow.
- GitHub webhook pipeline and CI policy gates.
- GAG/Copilot reconciliation loop.

## Phase R4 - Production Readiness (Sprints 7-8)

- Full ASDLC bundle automation.
- Observability and reliability hardening.
- Release operations and governance dashboards.
