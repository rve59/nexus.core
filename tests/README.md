# Test Suite Structure

This directory tracks test evidence with backlog linkage.

## Traceability Rules

1. Every test file must map to at least one Story ID.
2. Integration and workflow tests should map to Feature IDs.
3. Test results used for release gates must be referenceable from session logs.
